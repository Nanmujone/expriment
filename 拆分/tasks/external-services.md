# 外部服务模块任务清单

**目标：** 隔离网易云、OpenAI、本地媒体和 Windows 凭据能力，统一 DTO、超时、取消和错误。

**依赖：** `background-tasks-diagnostics` 的错误与追踪约定；不依赖业务数据库。

## 最小任务

- [ ] `EXT-01` 定义通用 `Result`、追踪 ID、标准错误码和脱敏规则。
- [ ] `EXT-02` 汇集并复用各消费领域定义的 `PlaylistProvider`、`AudioSourceProvider`、`LyricsProvider`、`AIProvider`、`MediaFileProvider`、`CredentialProvider` 端口，建立适配器契约测试套件；本模块不得重新定义这些端口。
- [x] `EXT-03` 建立独立网易云可行性验证分支/脚本，记录接口来源、许可、条款风险、测试日期和停止条件。
- [x] `EXT-04` 验证无需登录/Cookie 的公开歌单解析和最多 500 首元数据；失败则停止后续在线实现并提交需求确认。结果：失败，进入 `PARTIAL_OFFLINE`。
- [x] `EXT-05` 验证临时音频地址、流播放、Range 和跳转；不得实现下载、破解或权限绕过。结果：能力未通过，未继续实现。
- [x] `EXT-06` 验证逐行及可用时逐词歌词，并建立版权、地区、会员、网络和接口错误分类。结果：逐行可用、逐词未通过，不作为在线能力继续开发依据。
- [ ] `EXT-07` 建立 Node.js 辅助进程的 `/v1` HTTP/JSON Schema 和错误码契约。
- [ ] `EXT-08` 实现 Python 进程管理：高熵令牌经标准输入传递、回环随机端口、健康检查、有限重启和受控关闭。
- [ ] `EXT-09` 实现 Node.js 请求令牌校验；拒绝任意上游 URL、文件下载和原始 Cookie 返回。
- [ ] `EXT-10` 实现曲库、播放器和歌词领域所拥有的网易云端口适配器及契约测试；临时音频地址只存在内存。
- [ ] `EXT-11` 实现 OpenAI Chat Completions 兼容适配器，支持结构化输出探测、严格 JSON、取消和统一错误。
- [ ] `EXT-12` 实现 Windows Credential Manager 适配器；测试设置导出和日志均不包含密钥值。
- [ ] `EXT-13` 实现本地 MP3/LRC 只读关联和读取校验；任何清理接口只能删除关联记录。
- [ ] `EXT-14` 运行 Python/Node 契约测试及进程退出、超时、迟到响应和崩溃测试。

## 完成定义

- [x] 网易云验证门通过后才能继续依赖在线链路；失败时没有规避实现。
- [ ] Node 只监听 `127.0.0.1`，每次请求验证会话令牌且不访问业务数据库。
- [ ] 密钥、Cookie、临时地址和媒体路径不进入日志、数据库或备份。
