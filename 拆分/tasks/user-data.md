# 用户数据管理模块任务清单

**目标：** 管理学习数据清理、30 天恢复期以及版本化备份和原子恢复。

**依赖：** `persistence-migrations`、`phrase-favorites`、`ai-learning`、`background-tasks-diagnostics`。

## 最小任务

- [ ] `DATA-01` 定义临时缓存、问答、解析、收藏、曲库五类数据范围及删除预览 DTO。
- [ ] `DATA-02` 实现分项清理预览，返回记录数量、影响范围和不可删除的外部媒体说明。
- [ ] `DATA-03` 实现二次确认令牌；确认内容变化或令牌过期时拒绝执行清理。
- [ ] `DATA-04` 实现分项清理事务；测试任一步失败均完整回滚。
- [ ] `DATA-05` 实现全量本地数据清除；只删除应用数据和关联记录，绝不删除原 MP3/LRC。
- [ ] `DATA-06` 实现待清理项目查询、`RestorePendingData`、剩余天数和“已到期、等待再次确认”状态展示数据。
- [ ] `DATA-07` 实现到期后的二次确认删除：30 天到期只把项目转为“等待确认”，不得自动物理删除；用户再次确认后重新计算引用、检查恢复标记并校验短期确认令牌，任一条件变化即取消删除。
- [ ] `DATA-08` 定义版本化 ZIP 清单、Schema 版本、文件 SHA-256 和排除项。
- [ ] `DATA-09` 实现备份导出；导出前明确提示备份未加密、包含哪些个人学习数据且不会自动上传，用户确认后再执行；排除 API 密钥、Cookie、临时缓存、原始媒体和敏感诊断内容。
- [ ] `DATA-10` 实现备份校验；哈希错误、缺文件和不兼容版本必须在修改现有数据前失败。
- [ ] `DATA-11` 实现恢复到临时目录、临时迁移验证及数据库一致性检查。
- [ ] `DATA-12` 实现同卷原子替换与失败回滚；模拟替换失败并验证原数据不变。
- [ ] `DATA-13` 提供“升级前创建版本备份并返回已校验凭据”的应用用例；由应用集成模块把该用例接到迁移协调器，避免用户数据模块反向调用迁移；测试备份失败时不会产生有效凭据。
- [ ] `DATA-14` 运行 `pytest tests/user_data tests/backup -q`。
- [ ] `DATA-15` 由用户数据领域定义 `UserDataRepository`、`BackupArchivePort`、`AtomicDataReplacePort` 及行为契约；ZIP、文件系统和 SQLAlchemy 组件只实现端口。
- [ ] `DATA-16` 建立备份往返测试：在干净测试目录恢复后逐项比对歌单、歌曲关系、歌词、偏移、解析版本、收藏、问答和非敏感设置；同时验证密钥、Cookie、缓存和原始 MP3/LRC 未进入备份。
- [ ] `DATA-17` 对备份导出、临时恢复、原子替换和全量清理分别注入磁盘不足；验证停止当前操作、旧数据不变、临时文件可安全处理并显示所需空间与缓存清理入口。

## 完成定义

- [ ] 清理操作均有预览、二次确认和事务保护；30 天到期不会自动物理删除，必须由用户再次确认。
- [ ] 备份可校验、可恢复且不包含密钥和原始媒体。
- [ ] 所有删除和卸载路径都不会修改或删除用户原始 MP3/LRC。
