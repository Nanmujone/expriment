# 英文歌曲歌词学习播放器总体进度

> 本文件只勾选真实完成且已通过模块“完成定义”的项目。模块内进度以对应文件为准。实现任务直接采用 TDD：先写失败测试、再写最小实现、最后重构；不得把计划项或模拟结果当作完成证据。

## 推荐实施顺序

### 阶段 0：风险验证与工程地基

- [ ] 完成网易云在线链路可行性验证门（`EXT-03`～`EXT-06`）；未通过则暂停在线链路并回到需求确认。
- [ ] 完成[后台任务与诊断](./background-tasks-diagnostics.md)的任务、错误、取消、脱敏和磁盘异常基础。
- [ ] 完成[持久化与迁移](./persistence-migrations.md)的数据模型、领域仓储端口适配器、短事务、迁移预检和失败回滚。
- [ ] 完成[外部服务](./external-services.md)对领域端口的 Node、网易云、AI、凭据和本地媒体适配。

### 阶段 1：听歌主链路

- [ ] 完成[曲库与歌单](./playlist-library.md)的导入、缓存、刷新、删除和待清理标记。
- [ ] 完成[歌词](./lyrics.md)的解析、同步、偏移和重播边界。
- [ ] 完成[播放器](./player.md)的来源解析、队列、控制、媒体键和句子重播。
- [ ] 验证“公开歌单链接 → 缓存曲库 → 点击歌曲 → 5 秒目标内出声 → 同步歌词”主链路。

### 阶段 2：学习与数据闭环

- [ ] 完成[AI 学习](./ai-learning.md)的配置、整首解析、编辑、问答、质量门和错误隔离。
- [ ] 完成[词句收藏](./phrase-favorites.md)的选择、释义、保存和检索。
- [ ] 完成[用户数据管理](./user-data.md)的清理、30 天恢复、到期后二次确认、备份往返和恢复。
- [ ] 验证“选歌词 → AI 解析/追问 → 收藏词句 → 搜索编辑 → 备份恢复”闭环。

### 阶段 3：桌面体验与应用接线

- [ ] 完成[导航与页面](./navigation-pages.md)的页面、状态、快捷键、可访问性和托盘交互。
- [ ] 完成[应用集成与 UI 接线](./application-integration.md)的组合根、全部核心动作、隐私确认、启动/关闭和跨模块错误恢复。
- [ ] 验证 UI 只调用应用服务，领域拥有端口且适配器实现端口；架构依赖测试通过。
- [ ] 验证升级流程由应用层顺序编排“升级前备份 → 备份校验凭据 → 迁移”，不存在备份与迁移模块依赖环。

### 阶段 4：性能、打包与发布

- [ ] 使用固定公开歌单和原创 MP3/LRC 完成功能、AI 质量、备份往返、磁盘不足和逐条需求追踪验收。
- [ ] 完成缓存显示、100 首导入、首声、内存、歌词误差、后台可用性和 4 小时连续播放性能验收。
- [ ] 完成[打包与发布](./packaging-release.md)的 PyInstaller、Qt/Node 随包、安装升级卸载、敏感扫描、SBOM、标签和变更日志。
- [ ] 执行全部自动测试和 Windows 安装包烟雾测试，保存测试环境、构建版本、命令和结果；任何未在真实环境完成的项目保持未勾选。

## 跨模块里程碑

- [ ] `M0` 在线链路验证门通过，并记录接口来源、许可、风险、日期和停止条件。
- [ ] `M1` 应用能在无网络情况下于 5 秒内显示缓存曲库。
- [ ] `M2` 可从公开歌单播放在线歌曲，也可在用户确认合法用途后为受限歌曲关联只读本地 MP3/LRC。
- [ ] `M3` 歌词按能力降级，逐行/逐词同步、偏移和句子重播边界正确。
- [ ] `M4` AI 仅由用户主动触发；失败、取消和格式错误不影响播放或旧解析，20 行抽查质量门通过。
- [ ] `M5` 词句收藏、问答、编辑、搜索、筛选和删除形成完整学习闭环。
- [ ] `M6` 歌单删除、30 天恢复、到期后用户再次确认删除和共享歌曲保护通过事务测试；不存在到期自动物理删除。
- [ ] `M7` 备份往返逐项一致，升级前备份校验、临时恢复、迁移、回退和原子替换失败回滚全部通过，且模块无依赖环。
- [ ] `M8` 密钥、Cookie、临时音频地址、媒体路径和敏感正文未进入日志、数据库、备份、Git 历史或发布产物；发送、导出和删除均有隐私确认。
- [ ] `M9` 全部核心用户动作已从 UI 接到应用服务，Windows 启动、托盘、媒体键、关闭和故障恢复通过验收。
- [ ] `M10` 固定材料上的性能与稳定性目标全部通过，包含缓存 ≤5 秒、100 首 ≤30 秒、首声 ≤5 秒、内存 ≤700MB、歌词误差 ≤500ms 和 4 小时连续播放。
- [ ] `M11` Windows 安装包包含 Python、Qt Multimedia 和 Node.js 所需组件；全新安装、升级、回退、卸载及外部媒体保护通过验收。
- [ ] `M12` 需求第 5～12 节和第 14.2 节逐条追踪完整，自动测试与发布烟雾通过，SBOM、敏感扫描、校验值、版本标签和变更日志相互对应。
