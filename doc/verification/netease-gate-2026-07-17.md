# 网易云在线链路验证门报告（2026-07-17）

## 1. 结论

**门状态：`PARTIAL_OFFLINE`。**

固定公开歌单页面能够在不登录、不发送 Cookie 的条件下打开，歌词接口也能对本次抽样返回逐行时间轴；但完整曲目元数据、可安全跟随的临时音频地址、Range/进度跳转、逐词时间轴、稳定的受限原因信号以及官方授权/稳定性承诺均未同时成立。依据 HLD 13.1 和 D-001，网易云在线导入、播放和歌词适配器不得继续实现或启用。

当前仓库尚无在线业务入口，因此无需删除或隐藏现有入口。后续实现必须保持阻塞；本地播放、已有缓存阅读和 AI 学习等不依赖网易云在线链路的能力可继续。

这不是法律意见。当前证据只支持“没有确认到足以继续开发的公开官方 API 授权和稳定性合同”；如需重新开启在线链路，应先取得平台书面授权或完成独立法律/条款审查，再重新执行本门。

## 5. 错误分类

脚本定义并由离线单元测试覆盖以下稳定分类。只有上游给出明确限制信号时才判为版权、地区或会员；不能仅凭空 URL、HTTP 403 或播放失败进行猜测。

| 分类 | 可接受的明确证据 | 降级原则 |
| --- | --- | --- |
| `copyright` | `copyright`、`no_copyright`、`license_restricted` | 保留歌曲；显示版权限制；允许合法本地兜底 |
| `region` | `region`、`geo_blocked` | 保留歌曲；显示地区限制；不得代理或绕过 |
| `membership` | `membership`、`vip_required`、`fee_required` | 保留歌曲；显示会员限制；不得伪造登录或会员态 |
| `network` | 连接、DNS、socket 等传输异常 | 可有限重试；不改变本地数据 |
| `timeout` | 连接/读取/总超时 | 可有限重试；不把超时误判为版权限制 |
| `interface` | 非预期 HTTP、无效 JSON/Schema、必要字段缺失、非成功业务码 | 显示接口不可用；保留缓存；阻止在线入口 |

本次真实固定材料只观察到 `interface`（详情接口业务码 `20001`）。没有通过构造受保护内容、切换地区、登录会员或其他绕过方式获取版权/地区/会员现场样本；这三类仅完成确定性契约测试，**不声称真实上游映射已验证**。

## 8. 重新验证前置与停止条件

只有同时满足以下条件才可在新的验证分支重新评估：

1. 获得可核验的网易云官方 API 文档与对本使用方式的授权/许可依据；
2. 无登录、无 Cookie 获取固定歌单 0～500 首完整必要元数据；
3. 临时音频链路全程满足可接受的传输安全，Range 0 和非零偏移均返回可验证的分段响应；
4. 逐行歌词稳定，存在逐词源时能验证逐词结构，无逐词时能明确降级；
5. 上游提供可稳定映射的版权、地区、会员、网络和接口信号；
6. 重复、跨时段探测均通过，且没有登录、Cookie、下载、破解、替代源或隐藏开关。

在上述条件全部成立前，任何元数据业务码异常、HTTP 降级、Range 不成立、来源授权不明、字段结构变化或错误原因不可判定都应立即停止在线实现并保持 `PARTIAL_OFFLINE`。
