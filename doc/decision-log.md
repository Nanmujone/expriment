## 已采纳决策

| ID | 状态 | 决策 | 影响与验证要点 |
| --- | --- | --- | --- |
| D-001 | 已采纳 | 网易云验证门是在线能力的有条件开发前置；只有 `ONLINE_GATE_PASSED` 可启用在线入口，失败进入 `PARTIAL_OFFLINE`。 | 禁止 Cookie、会员绕过、隐藏开关或非授权替代源；本地/缓存/AI 非依赖能力可继续。 |
| D-002 | 已采纳 | 30 天仅是待清理等待期，到期后仍需用户针对当批数据再次明确确认才可物理删除。 | 到期任务只转为 `awaiting_purge_confirmation`；一次性授权不可跨批次或重启复用。 |
| D-003 | 已采纳 | AI 解析和问答固定绑定 `lyrics_document_id`、歌词版本与规范化内容 SHA-256。 | 歌词变化后旧结果显示为旧版；迟到响应不得写入新歌词上下文。 |
| D-004 | 已采纳 | ZIP 恢复先做结构、类型、限额、空间、哈希和版本预检，再流式写入隔离临时目录。 | 拒绝路径穿越、绝对/UNC 路径、重复路径、链接/reparse point、ZIP 炸弹和各种超限；失败不修改现有数据。 |
| D-005 | 已采纳 | 非回环 AI 地址强制 HTTPS、证书链及主机名校验；禁止携带认证信息跨源重定向；恢复后解绑旧凭据。 | 无“忽略证书”开关；默认不跟随重定向，仅可有限同源 HTTPS 重定向。 |
| D-006 | 已采纳 | 全量清除及卸载选择删除应用数据时，必须清除本应用 Windows 凭据。 | 外部 MP3/LRC 永不删除；凭据清除失败必须报告残留并允许重试。 |
| D-007 | 已采纳 | 发布产物必须带机器可读 SBOM 和第三方许可/NOTICE，并对全部传递依赖做许可审查。 | 未知许可、缺许可文本或不兼容许可阻止发布。 |
| D-008 | 已采纳 | 网易云、AI、LRC 和备份中的外部文本一律按不可信纯文本渲染。 | 禁止解释为 HTML/Markdown、富文本链接或 WebView 脚本。 |
| D-009 | 已采纳 | 播放默认使用在线来源；仅在线不可用或用户显式选择时使用本地文件。 | 来源状态可见且可切回在线，符合“本地是兜底而非主流程”。 |
| D-010 | 已采纳 | 普通暂停 `PausedByUser` 与句重播完成 `SegmentCompleted` 是不同领域状态。 | 普通继续从暂停位置；重播完成后的继续从句首。 |
| D-011 | 已采纳 | 增强型 LRC 逐词标记方言必须在详细设计形成白名单、冲突规则和逐方言测试语料。 | 未完成详细设计与契约测试前，不得宣称逐词解析开发完成；未知方言安全降级到逐行/纯文本。 |
| D-012 | 已采纳 | 临时音频缓存默认 TTL 不超过 24 小时、总配额不超过 512 MiB，超额按 LRU 清理并在启动时强制扫描。 | 退出清理仅是补充；缓存不进入备份、Git 或导出。 |
| D-013 | 已采纳 | 备份不保存本地媒体绝对路径、用户名目录、盘符、网络共享路径或凭据引用。 | 只保存显示名、内容指纹和待重新关联标记；恢复后重新选择媒体。 |
| D-014 | 已采纳 | Git 提交、测试、敏感信息扫描、SBOM/许可、安装包哈希、变更日志和版本标签组成可追溯发布证据链。 | 主分支可运行且所有发布门通过后才可创建版本标签。 |
| D-015 | 已采纳 | 保留 `gpt-5.6-terra` 作为已确认有效的 AI 质量验收基准模型。 | 真实 AI 验收仍要求用户提供可访问该模型的服务地址、API 密钥和有效权限；否则只能完成模拟/契约测试。 |
| D-016 | 已采纳 | 按用户 2026-07-17 的明确指令，不使用任何 `superpowers` 插件；直接执行 TDD、系统化根因诊断、独立需求审查和独立质量审查。 | 已从 `Doc1/prompt.md` 与任务文件移除插件依赖；质量门和审查职责不因此降低。 |
| D-017 | 已采纳 | 开发环境使用工作区隔离的 Python 3.12、`.venv` 和 `uv` 缓存/解释器目录，不改变系统 Python 3.14。 | `.tools/`、`.uv-cache/`、`.venv/` 均被 Git 忽略；`pyproject.toml`、`.python-version`、`uv.lock` 作为可复现基线提交。 |
| D-018 | 已采纳 | 用户接受首版以 `PARTIAL_OFFLINE` 交付，优先完成本地 MP3/LRC、歌词学习和 AI 主链路；不等待网易云在线接口恢复。 | 在线入口明确禁用并显示原因；不得用登录、Cookie 或非授权接口规避验证门。 |
